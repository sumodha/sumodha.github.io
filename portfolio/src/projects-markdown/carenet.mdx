# Introduction 
CareNet was built during **Rice Hacks under the Social Good track**, which challenged teams to identify a pressing social problem and build a technology-driven solution with empathy and justice at its core.

**Navigating the U.S. social welfare system is unnecessarily complex.** Between scattered resources, dense paperwork, and confusing eligibility requirements, the people who need help the most are often the ones least equipped to access it. **CareNet is an AI-powered agent that simplifies this process**, guiding users through discovering programs they qualify for and supporting them through the application process, all through a friendly, conversational experience.

# Problem
We noticed a significant gap in how people access the U.S. social welfare system. **Government paperwork is notoriously difficult to navigate, yet there are few tools designed to help people through the application process.** The communities that depend on these programs, low-income families, the elderly, individuals facing housing or food insecurity, often face the highest barriers to accessing them. We built **CareNet to bridge that gap**.

# Solution
CareNet is a **user-centered AI agent that guides users through discovering social welfare programs tailored to their needs and supports them in applying**. Through casual, empathetic conversation, the agent collects relevant information and cross-references it against program eligibility requirements, making the process feel less like filling out forms and more like talking to someone who wants to help.

# Key Features 
- **Conversational Eligibility Matching** — Parses user information through natural dialogue and cross-references it against program requirements.
- **Stochastic Eligibility Engine** — Uses probabilistic models to drive a dynamic question flow, narrowing down program matches with over 90% accuracy while minimizing the number of questions asked.
- **Empathetic Agent Design** — Structured to respond with sensitivity to each user's situation.
- **Live Program Data** — Web scrapes state and federal social welfare program databases to ensure eligibility information stays current and accurate.
