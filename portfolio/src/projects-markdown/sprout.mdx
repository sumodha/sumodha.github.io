# Introduction 
Sprout aims to **democratize mental wellness** by providing accessible, evidence-based Positive Psychology Interventions (PPIs) through a **seamless, user-friendly mobile app experience**. They strive to **empower individuals to cultivate lasting happiness and resilience**, fostering a healthier, more mindful world.

## WHY SPROUT?
I was first introduced to Sprout through a friend, who said that they were looking for software developers. As I explored their application and learned more about their mission, I was immediately drawn in. Mental health is something I'm actively working to improve in my own life, so **the opportunity to join a team dedicated to helping others care for theirs through daily activities and check-ins felt like the perfect fit**. It's a chance to grow my technical skills while deepening my own understanding of mental wellness. 

# Key Features 
- **User authentication and onboarding**
    - Email/password signup and login
    - Email verification flow for new accounts
    - JWT-based access and refresh tokens
    - Plan selection / SONA study onboarding path
- **Daily mental wellness tracking**
    - Daily mood check-in with happy/sad levels and notes
    - Activity completion logging
    - Emotion summary endpoints for analytics
- **Guided wellness activities**
    - Large library of structured exercises across many domains:
        - gratitude, strengths, values, self-compassion, flow, kindness, mindful listening, problem solving, SMART goals, forgiveness, gratitude letters, etc.
    - Multiple screens per activity for multi-step journeys, reflection, writing, video guidance, and completion tracking
- **Session and event tracking**
    - Start/end session endpoints
    - Session event logging
- **Push notifications and reminders**
    - Expo push token registration
    - Daily check-in reminder notification sender
    - Notification handling that navigates users to the check-in screen when tapped
    - Timezone-aware user settings
- **Mobile app UX / navigation**
    - Expo + React Native app with stack-based onboarding and activity flows
    - Bottom-tab navigation for main app sections
    - Cross-platform support via Expo

# Technical Contributions
## CI/CD PIPELINE
Designed and implemented an end-to-end continuous deployment pipeline using **GitHub Actions**, automating the full workflow from code commit to production deployment on a DigitalOcean droplet. The pipeline (`deploy.yml`) triggers on every push to the `main` branch, building a Docker image, streaming it to the remote server via SSH, and performing a zero-downtime container swap.

### Local Development Pipeline 
Separated local and production backend environments so team members could develop and test without deploying to production. Created documentation for local backend deployment.
